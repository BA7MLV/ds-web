您的终身学习空间｜DeepStudent

DeepStudent 是一款基于 Tauri 2 构建的 AI 原生、本地优先开源学习系统（AGPL-3.0）。它以「对话即入口」把智能对话、学习资源管理、RAG 检索、Anki 制卡与练习回顾串成可持续的学习闭环。

你可以用它：导入并管理资料（PDF/Word/图片…）→ 本地索引与检索 → 对话驱动阅读/调研/练习 → 一键制卡与间隔复习。

  • 本地优先：离线可用，数据可审计可备份，敏感信息加密存储
  • AI 入口统一：Chat V2 + Skills + MCP 工具生态 + CardForge 任务编排
  • 学习资料统一：VFS 单一数据源，全文检索 + 向量检索（RAG）
  • 学习闭环：题库练习、错题归档、知识导图、翻译工作台、Anki 制卡

常见问题（FAQ）

DeepStudent 是云服务吗？

不是。DeepStudent 以「本地优先」为核心，数据默认保存在本地。你可以选择配置第三方模型/搜索服务，但不会要求注册登录。

需要自备 API Key 吗？

可选。你可以配置自己的模型 API Key；同时也支持合作伙伴提供的免费模型，下载即可体验核心能力。

它和 Anki 的关系是什么？

DeepStudent 提供 CardForge 制卡任务：从文档/对话中提取知识点并生成卡片，支持导出 APKG 或通过 AnkiConnect 同步到 Anki。
