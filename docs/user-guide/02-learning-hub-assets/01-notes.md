# 笔记 (Notes)

## 概述

Deep Student 的笔记模块是一个功能强大的 Markdown 编辑器，旨在为你提供流畅的写作体验和深度集成的 AI 辅助能力。

**核心特性：**
- **所见即所得 (WYSIWYG)**：基于 Crepe 引擎，支持 Markdown 语法即时渲染。
- **自动保存**：编辑内容会自动保存（防抖 1.5秒），无需担心数据丢失。
- **AI 辅助编辑**：支持 AI 续写、润色、纠错，并通过 Diff 视图展示修改建议。
- **双向链接**：支持通过 `[[` 语法引用其他笔记或资源。
- **本地优先**：所有笔记以 Markdown 格式存储在本地 VFS 中。

---

## 编辑器使用

### 基础编辑

编辑器支持标准的 Markdown 语法，包括：
- **标题**：`# H1`, `## H2`, `### H3`
- **列表**：`-` 或 `*` (无序列表), `1.` (有序列表)
- **引用**：`> 引用内容`
- **代码块**：````python ... ````
- **表格**：Markdown 表格语法
- **数学公式**：`$E=mc^2$` (行内), `$$...$$` (块级)
- **图表**：支持 Mermaid 流程图、时序图等

### 插入资源

- **图片**：直接将图片拖拽到编辑器中，或使用 `/image` 命令。
- **链接**：使用 `[链接文本](URL)` 或直接粘贴 URL。
- **双链引用**：输入 `[[` 触发引用选择器，可链接到其他笔记、教材或题目集。

### 快捷操作 (Slash Command)

在空行输入 `/` 可唤起命令菜单，快速插入：
- 标题、列表、引用
- 表格、代码块、分割线
- 图片、数学公式

## 笔记管理

### 侧边栏导航

左侧侧边栏提供了笔记的树状管理视图：

- **新建笔记**：点击顶部工具栏的 "文件" 图标。
- **新建文件夹**：点击顶部工具栏的 "文件夹" 图标。
- **拖拽排序**：支持拖拽笔记移动到文件夹，或调整排序。
- **右键菜单**：支持重命名、删除、添加到收藏、引用教材/题目集。

### 收藏与筛选

- **收藏**：点击侧边栏工具栏的 "星标" 按钮，仅显示收藏的笔记。
- **搜索**：在侧边栏搜索框输入关键词，快速过滤笔记标题。

## AI 辅助写作

Deep Student 将 AI 能力深度集成到了编辑器中。

### 触发方式
1. 选中一段文本，点击弹出的 **"AI 编辑"** 按钮。
2. 或在对话侧边栏中，对笔记内容发出修改指令（如 "润色这段话"）。

### Diff 视图
当 AI 对笔记提出修改建议时，编辑器会进入 **Diff 模式**：
- **红色背景**：表示 AI 建议删除的内容。
- **绿色背景**：表示 AI 建议新增的内容。
- **操作栏**：你可以选择 **"接受 (Accept)"** 或 **"拒绝 (Reject)"** 修改。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + S` | 手动保存（虽然支持自动保存，但手动保存更安心） |
| `Cmd/Ctrl + Z` | 撤销 |
| `Cmd/Ctrl + Shift + Z` | 重做 |
| `Cmd/Ctrl + B` | 粗体 |
| `Cmd/Ctrl + I` | 斜体 |
| `/` | 唤起插入菜单 |
