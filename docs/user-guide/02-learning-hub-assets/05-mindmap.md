# 知识导图 (Mind Map)

## 概述

**知识导图** 是 Deep Student 提供的可视化知识整理工具。它允许你以结构化的方式捕捉灵感、整理笔记和梳理复杂概念。

**核心特性：**
- **双向视图**：支持在 **思维导图 (Canvas)** 和 **大纲 (Outline)** 视图之间无缝切换。
- **多种布局**：提供思维导图、逻辑图、组织结构图等多种专业布局。
- **键盘优先**：全键盘操作支持，无需鼠标即可快速构建导图。
- **样式定制**：内置多种精美主题，支持节点样式微调。
- **格式兼容**：支持 OPML、Markdown、JSON 等通用格式的导入导出。

---

## 视图模式

### 思维导图视图
这是默认的图形化视图，适合进行发散性思维和整体结构把握。
- **缩放与移动**：使用鼠标滚轮缩放，按住空格键或鼠标中键拖动画布。
- **节点折叠**：点击节点连接线上的圆点可折叠/展开子节点。

### 大纲视图
以线性列表的方式展示内容，适合快速录入大量文本或进行长篇内容的梳理。
- **层级调整**：使用 `Tab` / `Shift + Tab` 调整缩进。
- **拖拽排序**：直接拖拽列表项调整顺序。

## 操作指南

### 节点操作 (快捷键)

| 操作 | Windows/Linux | macOS | 说明 |
|------|---------------|-------|------|
| **添加子节点** | `Tab` | `Tab` | 在当前节点下创建子节点 |
| **添加同级节点** | `Enter` | `Enter` | 在当前节点下方创建同级节点 |
| **编辑内容** | `Space` / 双击 | `Space` / 双击 | 进入编辑模式 |
| **换行** | `Shift + Enter` | `Shift + Enter` | 在节点内容中换行 |
| **完成编辑** | `Enter` / `Esc` | `Enter` / `Esc` | 退出编辑模式 |
| **删除节点** | `Delete` | `Delete` | 删除选中节点及其子节点 |
| **撤销** | `Ctrl + Z` | `Cmd + Z` | 撤销上一步操作 |
| **重做** | `Ctrl + Shift + Z` | `Cmd + Shift + Z` | 重做 |
| **搜索** | `Ctrl + F` | `Cmd + F` | 在导图中搜索内容 |

### 布局与结构

点击工具栏的 **"结构"** 按钮，可以选择不同的布局方式：

- **思维导图**：
  - **平衡图**：根节点在中心，子节点向四周发散。
  - **树形图 (向右/向左)**：经典的树状结构。
- **逻辑图**：
  - **逻辑图 (向右/向左)**：使用直角连接线，结构更严谨。
- **组织结构图**：
  - **向下/向上**：适合展示层级汇报关系。

### 样式定制

点击工具栏的 **"样式"** 按钮：
- **全局主题**：选择预设的配色方案（如极简白、深色模式、彩色等）。
- **节点样式**：自定义当前选中节点的文字颜色、背景色、边框等。

## 导入与导出

### 导出
点击工具栏的 **"导出"** 按钮，支持以下格式：
- **图片**：PNG (透明背景)、SVG (矢量图)。
- **文档**：Markdown (文本大纲)、OPML (通用导图格式)、JSON (数据备份)。

> **提示**：导出图片时，请先切换到思维导图视图。

### 导入
支持导入 OPML、Markdown 或 JSON 文件，快速还原导图结构。

## 与 AI 协作

### 引用到对话
在 Chat V2 对话中，你可以通过引用知识导图，让 AI：
- **续写**：基于当前导图结构，生成更多的子节点建议。
- **总结**：将导图内容总结为一篇文章。
- **转换**：将导图转换为 Anki 卡片或题目。

### AI 生成 (Mindmap Tools)
通过激活 `mindmap-tools` 技能，你可以直接用自然语言指令创建或修改导图（例如："帮我创建一个关于光合作用的思维导图"）。
