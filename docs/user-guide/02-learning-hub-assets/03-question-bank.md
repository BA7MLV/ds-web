# 题目集 (Question Bank)

## 概述

**智能题目集 (Question Bank)** 是 Deep Student 用于管理所有练习题、试卷和错题的核心模块。它不仅是一个静态的存储库，更是一个支持 **OCR 识别**、**智能拆题**、**AI 解析**、**复习计划** 和 **多维度统计** 的学习引擎。

## 核心功能

- **OCR 识别与拆题**：上传试卷照片，自动识别并拆分为单道题目。
- **智能解析**：AI 自动生成题目解析、知识点标签和难度评级。
- **复习管理**：支持题目状态标记（待复习/已掌握），可一键筛选复习题。
- **练习模式**：提供限时训练、模拟考试和每日一练。
- **多维度统计**：可视化展示学习趋势、活跃度和薄弱知识点。
- **CSV 导入导出**：方便与其他工具交换数据。

## 题目管理

### 创建题目

1. **手动创建**：
   - 点击侧边栏工具栏的 **"新建题目"**（文件图标）。
   - 填写题干、选项（如果是选择题）、答案和解析。
   - 设置题型、难度和标签。

2. **拍照识别 (OCR)**：
   - 点击 **"新建题目集"**（试卷图标）。
   - 上传试卷图片，系统会自动进行 OCR 识别。
   - 使用 **AI 拆题** 功能，将整张试卷自动分割为单道题目。

### 列表视图与筛选

题目列表采用极简的 Notion 风格设计，支持：
- **视图切换**：网格视图（卡片）/ 列表视图。
- **筛选**：按状态（新题/进行中/已掌握/复习）、难度、题型或收藏状态筛选。
- **搜索**：支持全文搜索（包含题干、解析和 OCR 文本）。
- **批量操作**：长按或点击编辑模式按钮，可批量删除或重置进度。

### 题目详情与编辑

点击题目卡片进入详情页/内联编辑模式：
- **做题**：输入答案，系统自动判断对错（主观题需自我评估）。
- **AI 辅助**：点击 **"AI 解析"** 获取详细解题思路。
- **编辑**：修改题目内容、标签或备注。

## 练习模式

Deep Student 提供多种练习模式，满足不同场景需求：

### 1. 顺序/随机练习
在题目列表页，点击 **"开始练习"** 按钮：
- **顺序练习**：按题目创建顺序进行。
- **随机练习**：随机抽取题目。
- **复习优先**：优先练习状态为 "待复习" 的题目。

### 2. 限时训练
设定时间和题量，模拟高压环境下的答题速度。
- 结束后自动生成统计报告，分析正确率和用时。

### 3. 模拟考试
基于标签、难度和题型分布，自动生成一套模拟试卷。
- 支持全真模拟，包含倒计时和答题卡。
- 考后生成详细的 **成绩单**，包含知识点掌握度分析。

### 4. 每日一练
系统根据你的学习历史，自动推荐一组题目（包含新题和复习题）。
- 完成每日打卡，保持学习连贯性。

## 统计分析

点击侧边栏的 **"统计"** 图标，查看你的学习数据：

- **学习趋势**：近期的做题数量和正确率曲线。
- **活跃热力图**：类似 GitHub 的贡献图，直观展示每日学习强度。
- **知识点掌握度**：按标签展示各知识点的掌握情况（已掌握/学习中/未开始）。
- **打卡日历**：查看每月的打卡记录和连续学习天数。

## 数据导入与导出

### CSV 导入
支持从 CSV 文件批量导入题目。
1. 点击 **"导入"** 按钮。
2. 上传 CSV 文件。
3. 配置字段映射（将 CSV 列映射到题干、答案等字段）。
4. 选择去重策略（跳过/覆盖/合并）。

### CSV 导出
将题目导出为 CSV 格式，方便在 Excel 中查看或迁移数据。
- 支持按筛选条件导出。
- 可选择导出的字段。

## 常见问题

### Q: 如何将错题加入复习计划？
A: 在做题时，如果答错或觉得题目重要，可以标记为 **"收藏"** 或手动将状态设为 **"待复习"**。系统会自动将其纳入后续的复习推荐中。

### Q: OCR 识别不准确怎么办？
A: 
1. 确保图片清晰、光线充足。
2. 在设置中切换更强大的 OCR 引擎（如 DeepSeek-VL）。
3. 识别后可手动修正文本内容。

### Q: 题目集和单个题目的关系？
A: "题目集" (Exam Sheet) 通常对应一张试卷或一次练习，包含多个 "题目" (Question)。你可以管理整个题目集，也可以在 "所有题目" 视图中管理单道题目。
