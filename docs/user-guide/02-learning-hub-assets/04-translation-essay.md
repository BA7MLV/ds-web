# 翻译与作文 (Translation & Essay)

## 概述

Deep Student 提供了专业的 **AI 辅助语言学习** 模块，分为 **翻译工作台 (Translation Workbench)** 和 **作文批改工作台 (Essay Grading Workbench)**。这两个工具都深度集成了大语言模型的能力，旨在提供不仅是结果，更是“教学”的体验。

---

## 翻译工作台

翻译工作台不仅提供文本翻译，还支持多引擎对比、流式响应和语音朗读。

### 主要功能

- **多模式输入**：支持文本输入、图片 OCR（截图/拍照）、文档上传（PDF/Word）。
- **流式翻译**：实时显示翻译过程，支持打断。
- **多引擎支持**：可以使用不同的 LLM 进行翻译，对比效果。
- **正式度调整**：一键切换 "正式 (Formal)"、"休闲 (Casual)" 或 "自动 (Auto)" 风格。
- **TTS 朗读**：高质量的语音朗读功能，辅助听力练习。
- **历史记录**：所有翻译记录自动保存到 VFS，随时查阅。

### 使用流程

1. **输入源文本**：直接输入，或拖拽图片/文档到输入区。
2. **配置参数**：
   - 选择源语言和目标语言（支持 Auto 检测）。
   - 设置正式度（可选）。
   - 编辑 Custom Prompt（可选，如“请翻译成地道的口语”）。
3. **开始翻译**：点击翻译按钮（或 `Cmd/Ctrl + Enter`）。
4. **后续操作**：
   - **编辑**：如果觉得 AI 翻译不完美，可以直接修改译文。
   - **朗读**：点击喇叭图标听发音。
   - **复制/导出**：一键复制或导出为文本文件。
   - **评分**：为 AI 的翻译质量打分（1-5星）。

## 作文批改工作台

作文批改工作台模拟了真人老师的批改流程，提供多维度的评分和建议。

### 主要功能

- **多维度评分**：从词汇、语法、结构、逻辑等多个维度进行打分。
- **流式批改**：AI 会像老师一样，逐步输出评语和修改建议。
- **多轮迭代**：支持对同一篇作文进行多次修改和重新批改，记录进步轨迹。
- **学段适配**：支持设置不同学段（初中、高中、大学、雅思/托福等），AI 会调整评分标准。
- **OCR 支持**：支持手写作文拍照识别，直接批改。

### 使用流程

1. **提交作文**：
   - 输入作文标题。
   - 粘贴作文内容，或上传手写照片（OCR）。
2. **配置要求**：
   - **作文类型**：议论文、记叙文、书信等。
   - **学段**：选择适合的学段标准。
   - **自定义要求**：如“请重点检查时态错误”。
3. **开始批改**：点击批改按钮。
4. **查看报告**：
   - **总体评价**：总分和简评。
   - **维度分析**：雷达图展示各维度得分。
   - **逐句点评**：具体的修改建议和替代表达。
5. **修改迭代**：
   - 根据建议修改作文。
   - 点击 **"开始下一轮"**，提交修改后的版本再次批改。
   - 系统会自动保存每一轮的历史记录，形成进步曲线。
