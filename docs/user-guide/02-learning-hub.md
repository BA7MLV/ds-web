# 学习资源

## 概述

学习资源（Learning Hub）是 Deep Student 的知识管理中心，采用访达（Finder）风格的文件管理界面，统一管理所有类型的学习材料。

**核心特性**：
- **访达风格文件管理**：文件夹树 + 内容列表，支持拖拽、多选、批量操作
- **多标签页**：同时打开多个资源，LRU 策略自动淘汰最久未访问的标签页
- **分屏查看**：将一个标签页固定到右侧，与左侧主视图并排对比
- **双视图模式**：列表视图（详细信息）/ 图标视图（缩略图网格）
- **与对话打通**：资源可直接引用到 Chat V2 对话上下文，或由 AI 工具读写

---

## 快速开始

### 进入学习资源

1. 在应用侧边栏点击「学习资源」图标（毕业帽）
2. 系统显示学习资源全屏界面

### 创建第一个笔记

1. 点击工具栏的 **新建** 按钮（或使用 `Cmd/Ctrl + N`）
2. 在弹出菜单中选择「新建笔记」
3. 系统自动创建新笔记并在右侧打开编辑器

### 导入教材

1. 点击 **新建** 按钮，选择「导入教材」
2. 在文件选择对话框中选择文档（支持 PDF、Word、Excel、PPT 等）
3. 等待导入完成，系统自动进行 OCR 识别和向量化索引

## 界面布局

### 桌面端

```
┌─────────────────┬──────────────────────────────────────┐
│  快捷访问 / 分  │  工具栏（搜索、视图切换、新建）      │
│  类导航（左侧） ├──────────────────────────────────────┤
│                 │  标签栏（多标签页）                  │
│  文件夹树       ├──────────────────────────────────────┤
│                 │  内容区（文件列表 或 打开的资源）    │
└─────────────────┴──────────────────────────────────────┘
```

- **左侧**：快捷访问（智能文件夹视图）+ 自定义文件夹树
- **右侧上方**：工具栏（搜索、视图切换、新建）+ 标签栏
- **右侧内容区**：文件列表 或 打开的资源内容

左侧侧边栏可折叠（点击折叠按钮或拖拽分隔线）。

### 移动端（三屏滑动布局）

移动端采用三屏滑动布局，通过手势左右滑动切换：

- **左屏**：应用类型入口（快捷导航）
- **中屏**：文件列表视图（默认显示）
- **右屏**：打开的资源内容

向右滑动进入左屏（应用入口），向左滑动打开资源后进入右屏（资源内容）。

## 视图与过滤

### 视图模式

点击工具栏中的视图切换图标在两种模式间切换：

- **列表视图**：显示文件名、类型、更新时间等详细信息
- **图标视图**：以网格形式显示缩略图，适合浏览图片和笔记

### 数据视图

- **文件夹视图**（默认）：按文件夹层级浏览，显示文件夹中的资源
- **资源浏览视图**：忽略文件夹结构，按资源类型过滤浏览所有资源

### 资源类型过滤

在资源浏览视图中，可按类型筛选：全部 / 笔记 / 教材 / 题目集 / 翻译 / 作文 / 知识导图 / 图片 / 文件

### 排序

支持按以下字段排序（升序/降序）：
- 更新时间（默认，最新优先）
- 创建时间
- 标题（字母顺序）

## 多标签页

多标签页允许同时打开多个资源，无需来回切换：

- 点击资源打开后，在标签栏显示标签页
- **LRU 淘汰**：超过最大标签页数时，自动关闭最久未访问的标签页
- 点击标签页的 × 关闭，或右键 → **关闭**

### 分屏查看

将一个标签页分屏到右侧与主视图并排：
1. 右键标签页 → **在右侧打开**
2. 左侧显示当前活跃标签页，右侧显示分屏标签页
3. 右键右侧标签页 → **关闭分屏** 退出分屏模式

## 文件操作

### 新建资源

点击工具栏的 **新建** 按钮（常用快捷键：`Cmd/Ctrl + N` 新建笔记，`Cmd/Ctrl + Shift + N` 新建文件夹）：

| 操作 | 说明 |
|------|------|
| 新建文件夹 | 创建新文件夹用于组织资源 |
| 新建笔记 | 创建空白 Markdown 笔记 |
| 新建题目集 | 创建空白题目集，可拍照识别题目 |
| 导入教材 | 选择本地文档导入为教材 |
| 新建翻译 | 创建空白翻译练习 |
| 新建作文 | 创建空白作文批改任务 |
| 新建知识导图 | 创建空白思维导图 |

### 文件操作（右键菜单）

右键点击文件或文件夹，可进行：
- **重命名**：修改资源名称
- **移动到**：将资源移动到其他文件夹
- **收藏 / 取消收藏**：标记常用资源
- **引用到对话**：将资源作为上下文引用到当前聊天会话
- **删除**：移入回收站（软删除）

### 多选操作

- 点击工具栏的**多选**按钮进入多选模式
- 进入多选模式后，点击文件逐个选中；`Ctrl/Cmd + A` 全选当前目录所有文件
- 选中后可批量移动、删除

### 回收站

- 删除的资源移入回收站，不立即永久删除
- 点击左侧快速访问中**「回收站」**条目查看已删除资源
- 可 **恢复** 或 **永久删除**

## 快速访问

左侧导航栏提供快速访问入口（智能文件夹），分为四个分组：

**快速导航**
- **桌面**：自定义快捷方式和应用入口
- **全部文件**：以真实文件夹结构浏览所有资源
- **最近**：最近访问的资源
- **收藏**：所有已加星标收藏的资源

**资源类型**
- **全部笔记** / **全部教材** / **全部题目集**
- **全部作文** / **全部翻译** / **知识导图**

**媒体**
- **全部图片** / **全部文档**

**系统**
- **回收站**：已删除的资源
- **向量化状态**：查看和管理向量索引
- **记忆管理**：AI 长期记忆存储

点击任意条目直接切换到对应的智能视图，无需手动导航文件夹。

## 搜索

工具栏搜索框支持实时搜索：

- 输入关键词后 300ms 自动触发搜索（防抖）
- 在当前目录下搜索文件名
- 搜索结果实时高亮显示
- 点击搜索结果直接打开对应资源

## 引用到对话（VFS 上下文注入）

学习资源中的任意文件都可以引用到 Chat V2 对话作为上下文：

**方式一：从学习资源中引用**
- 右键资源 → **引用到对话**
- 或选中资源后点击操作栏的 **引用** 按钮

**方式二：从对话中选择**
- 在对话输入栏点击 **资源库** 按钮（回形针图标）
- 在弹出面板中浏览或搜索，选中后点击添加

引用后，资源以标签形式显示在输入框上方，发送消息时 AI 读取并参考资源内容。

## 向量化状态

部分功能（RAG 检索、语义搜索）依赖向量索引。在资源列表中可查看向量化状态：

- **已完成**：资源已建立向量索引，可被 AI 检索
- **处理中**：正在建立向量索引
- **待处理**：等待向量化（通常自动处理）
- **失败**：向量化失败，可手动重试

**召回测试**：点击资源详情中的 **召回测试** 按钮，输入测试查询词，验证向量索引效果。

## 记忆管理

AI 在对话中产生的长期记忆以笔记形式存储在学习资源中。

### 首次设置

1. 在左侧导航找到 **记忆管理** 入口
2. 选择或创建一个文件夹作为记忆根目录
3. 系统将记忆以笔记形式存储在该文件夹中

### 管理记忆

- 搜索、查看、编辑或删除记忆条目
- 配合 `deep-student` 技能，AI 会主动调用和组织这些长期记忆
