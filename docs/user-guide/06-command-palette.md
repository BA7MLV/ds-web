# 命令面板与快捷键

## 概述

命令面板是 Deep Student 的效率中心，通过键盘即可触达应用的所有功能。

**快捷键**：
- **macOS**: `Cmd + K`
- **Windows/Linux**: `Ctrl + K`

---

## 常用命令

### 全局导航
| 命令 | 说明 |
|------|------|
| `跳转到智能对话` | 进入 Chat V2 主界面 |
| `跳转到学习中心` | 打开文件管理器 (VFS) |
| `跳转到制卡任务` | 进入制卡任务看板 |
| `跳转到模板库` | 管理 Anki 卡片模板 |
| `跳转到技能管理` | 管理 Skills |
| `跳转到设置` | 打开系统设置 |

### 对话操作
| 命令 | 说明 |
|------|------|
| `新建对话` | 创建一个新的聊天会话 |
| `新建题目分析` | 上传图片进行 OCR 识别和题目分析（`Cmd/Ctrl+Shift+A`） |
| `停止生成` | 停止 AI 响应生成 |
| `重新生成` | 重新生成上一条 AI 回复 |
| `清空对话` | 清空当前对话内容 |
| `选择AI模型` | 切换当前对话使用的 AI 模型 |
| `切换学习模式` | 切换学习辅导模式 |
| `导出对话` | 导出对话为 Markdown 或 JSON |

### 资源管理
| 命令 | 说明 |
|------|------|
| `新建笔记` | 创建一篇新的 Markdown 笔记 |
| `新建文件夹` | 在学习资源中新建文件夹 |
| `搜索笔记` | 在笔记侧边栏中搜索 |
| `全局搜索` | 全局搜索所有笔记、对话记录 |
| `打开翻译工具` | 启动 AI 翻译工作台 |
| `打开作文批改` | 启动 AI 作文批改工具 |

---

## 自定义快捷键

你可以为常用命令绑定个性化快捷键：

1. 打开 **设置 -> 快捷键**。
2. 找到目标命令。
3. 点击编辑按钮，按下你想要的组合键。
