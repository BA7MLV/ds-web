# 命令面板与快捷键

## 概述

命令面板是 Deep Student 的效率中心，通过键盘即可触达应用的所有功能。

**快捷键**：
- **macOS**: `Cmd + K`
- **Windows/Linux**: `Ctrl + K`

---

## 常用命令

### 全局导航

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `跳转到智能对话` | `Cmd/Ctrl + 1` | 进入 Chat V2 主界面 |
| `跳转到学习中心` | `Cmd/Ctrl + Shift + 7` | 打开文件管理器 (VFS) |
| `跳转到制卡任务` | `Cmd/Ctrl + Shift + 6` | 进入制卡任务看板 |
| `跳转到模板库` | | 管理 Anki 卡片模板 |
| `跳转到技能管理` | | 管理 Skills |
| `跳转到 PDF 阅读器` | | 阅读和标注 PDF 文档 |
| `跳转到仪表盘` | `Cmd/Ctrl + 5` | 数据统计与概览 |
| `跳转到数据管理` | `Cmd/Ctrl + E` | 数据导入导出与备份 |
| `跳转到设置` | `Cmd/Ctrl + ,` | 打开系统设置 |

### 对话操作

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `新建对话` | `Cmd/Ctrl + N` | 创建一个新的聊天会话 |
| `新建题目分析` | `Cmd/Ctrl + Shift + A` | 上传图片进行 OCR 识别和题目分析 |
| `停止生成` | `Cmd/Ctrl + .` | 停止 AI 响应生成 |
| `重新生成` | `Cmd/Ctrl + R` | 重新生成上一条 AI 回复 |
| `清空对话` | | 清空当前对话内容 |
| `选择AI模型` | | 切换当前对话使用的 AI 模型 |
| `导出对话` | | 导出对话为 Markdown 或 JSON |
| `导入对话` | | 从文件导入历史对话 |
| `从此处分支` | | 基于当前消息创建新的分支会话 |
| `复制上一条回复` | `Cmd/Ctrl + Shift + C` | 复制最后一条 AI 回复 |
| `收藏对话` | | 将当前对话加入收藏 |

### 模式切换

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `切换 RAG 模式` | `Cmd/Ctrl + Shift + R` | 开启/关闭检索增强生成 |
| `切换联网搜索` | `Cmd/Ctrl + Shift + W` | 开启/关闭实时联网搜索 |

### 全局功能

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `全局搜索` | `Cmd/Ctrl + P` | 搜索笔记、对话、文档 |
| `切换主题` | `Cmd/Ctrl + Shift + T` | 切换明暗主题 |
| `切换全屏` | `F11` | 切换全屏显示模式 |
| `放大界面` | `Cmd/Ctrl + =` | 增大界面缩放 |
| `缩小界面` | `Cmd/Ctrl + -` | 减小界面缩放 |
| `重置缩放` | `Cmd/Ctrl + 0` | 恢复默认缩放 |

### 资源管理

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `新建笔记` | | 创建一篇新的 Markdown 笔记 |
| `新建文件夹` | | 在学习资源中新建文件夹 |
| `搜索笔记` | | 在笔记侧边栏中搜索 |
| `打开翻译工具` | `Cmd/Ctrl + T` | 启动 AI 翻译工作台 |
| `打开作文批改` | | 启动 AI 作文批改工具 |

---

## 自定义快捷键

你可以为常用命令绑定个性化快捷键：

1. 打开 **设置 -> 快捷键**。
2. 找到目标命令。
3. 点击编辑按钮，按下你想要的组合键。
