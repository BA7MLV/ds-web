# Deep Student 快速入门

欢迎使用 Deep Student —— 一款 **AI 原生** 的终身学习工作台。

它不仅仅是一个聊天机器人，更是一个深度集成了 **知识管理 (VFS)**、**智能制卡 (Anki)** 和 **技能系统 (Skills)** 的完整学习环境。

---

## 核心理念

- **本地优先**：你的数据（笔记、对话、向量索引）完全存储在本地，隐私安全。
- **数据互通**：聊天记录、笔记、教材不再是孤岛，通过 VFS 相互引用和流转。
- **技能驱动**：通过激活不同的 "技能"，让 AI 瞬间变身为苏格拉底导师、制卡专家或科研助手。

---

## 界面概览

### 桌面端布局

- **左侧导航栏**：
    - 智能对话：Chat V2 主界面
    - 学习资源：VFS 文件管理器，管理笔记、教材、题目集
    - 技能管理：管理和自定义 AI 技能
    - 制卡任务：CardForge 制卡任务看板
    - 模板管理：Anki 卡片模板设计与管理
    - 设置：系统配置与数据治理
- **命令面板**：按 Cmd/Ctrl + K 随时唤起，快速执行任何操作

---

## 第一步：配置 AI 能力

Deep Student 需要接入 AI 模型才能工作。我们强烈推荐使用 **SiliconFlow (硅基流动)** 进行一键配置，它提供了极高性价比的推理模型。

1. 进入 **设置 -> 模型服务**。
2. 选择 **SiliconFlow** 供应商。
3. 输入你的 API Key。
4. 点击 **"一键分配"**。
   - 系统自动创建以下模型配置并完成分配：
     - **对话模型**：`deepseek-ai/DeepSeek-V3.2`（支持混合推理/思维链）
     - **Anki 制卡模型**：`Qwen/Qwen3-30B-A3B-Instruct-2507`
     - **嵌入模型**：`BAAI/bge-m3`（RAG 知识库）
     - **重排序模型**：`BAAI/bge-reranker-v2-m3`
     - **翻译模型**：`tencent/Hunyuan-MT-7B`
     - **OCR 引擎**：`PaddlePaddle/PaddleOCR-VL-1.5`（免费，支持 109 种语言）

> **提示**：你也可以手动配置 OpenAI、Anthropic、Google 等其他供应商。请务必在 **"模型分配"** 页面确保 "对话模型" 已正确选择，否则无法发送消息。

---

## 第二步：体验技能系统

Deep Student 的强大之处在于 **技能 (Skills)**。

> **注意**：`deep-student`（深度学者）技能在每个新会话中**自动开启**，AI 会主动记忆你的偏好并在后续对话中个性化回答。以下演示如何手动激活其他技能：

1. 回到 **智能对话** 页面。
2. 点击输入框下方的 **技能按钮**（闪电图标）。
3. 选择 **导师模式** (`tutor-mode`)。
4. 试着问一个问题，例如："帮我解释一下量子纠缠"。
5. 你会发现 AI 不再直接给答案，而是像导师一样循循善诱。

尝试其他技能：
- **ChatAnki 制卡** (`chatanki`)：把对话内容变成 Anki 卡片。
- **知识检索** (`knowledge-retrieval`)：让 AI 查阅你的本地笔记回答问题。

---

## 第三步：管理学习资源

1. 进入 **学习资源** 页面。
2. 点击 **新建** 按钮，**新建笔记** 或 **导入教材** (PDF)。
3. 在笔记中记录你的学习心得。
4. 回到对话页面，在输入栏点击 **资源库** 按钮，选中你刚才创建的笔记。
5. 现在，AI 已经"读"过了你的笔记，你可以针对笔记内容提问了！
