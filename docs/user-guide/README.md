## 功能概览

### 智能对话
- 多会话管理：分组管理、回收站、浏览模式
- 分析模式会话：专为图片 OCR 和题目识别
- 对话控制面板：温度、Top-P、频率惩罚、检索条数等参数按会话覆盖
- RAG 知识库检索、MCP 工具扩展
- 多模型并行对比、对话分支

### 技能系统
- 预配置 AI 行为模式（导师模式、ChatAnki 等）
- 按需加载工具组（知识检索、笔记、Anki 等）
- 三级加载：内置 → 全局 → 项目级
- 用户自定义技能（SKILL.md 格式）

### 学习资源
- 访达风格文件管理，支持多标签页和分屏查看
- 笔记（Markdown 编辑器）、教材（PDF 阅读器）
- 题目集（AI 识别）、翻译、作文批改、知识导图
- DOCX／XLSX／PPTX／音频／视频文件预览
- 向量化索引管理、回收站

### Anki 智能制卡
- 多格式文档解析（PDF、图片、手写、Markdown）
- 自动路由（simple_text / vlm_light / vlm_full）
- 任务控制（暂停、恢复、取消、重试）
- APKG 导出 / AnkiConnect 同步
- 多模板支持、Template Designer

### 数据管理
- 分层备份策略
- 自动备份
- WebDAV / S3 云同步
- 数据治理面板
