# Docs 简约风格设计方案

日期：2026-02-06

## 目标
- 将文档站点整体风格优化为“简约、克制、易读”。
- 保持品牌识别度与可读性，减少装饰性视觉噪音。
- 在不改变信息结构的前提下，降低玻璃感与强烈渐变。

## 设计原则
- 浅灰白基底、弱对比边界、轻量阴影。
- 视觉层级由字号/字重/留白建立，而非重度阴影或强对比色。
- 交互状态只做轻微明暗变化。

## 视觉与排版方案
- 背景改为纯色（浅灰/深灰），去除多层渐变与强背景纹理。
- 顶栏/侧边栏/正文区域改为实体色，移除模糊与玻璃感。
- 正文卡片弱化边框与圆角，统一为轻量 1px 线条与小圆角。
- 正文文字采用较柔和的深灰，标题稍加权重。
- 链接保留品牌蓝，但降低饱和度；悬停时出现细线强调。
- 代码块、引用、表格背景统一为浅灰底，圆角收敛。

## 实施范围
- 仅调整 `docs/.vitepress/theme/custom.css` 中的颜色变量、背景、阴影、卡片与组件样式。
- 不改动页面内容与结构。

## 风险与回滚
- 风险：对比度不足导致可读性下降。
- 回滚：恢复 `custom.css` 变量与相关组件样式即可。

## 验收与检查
- 首页与正文页背景无明显纹理干扰。
- 侧边栏、导航栏视觉干净、层级清晰。
- 正文排版清晰、链接与代码块易辨识。
- 亮/暗色模式均保持统一简约风格。
